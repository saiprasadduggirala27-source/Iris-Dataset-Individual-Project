# Random Forest Classification using Iris Dataset

## Project Overview
This project applies the **Random Forest Classification algorithm** to the **Iris Dataset** to classify iris flowers into three species:

- Setosa
- Versicolor
- Virginica

The project includes:
- data loading
- exploratory data analysis
- class distribution visualization
- correlation heatmap
- model training
- prediction and evaluation
- feature importance analysis
- accuracy comparison

---

## Aim
To build a Random Forest Classification model using the Iris Dataset and evaluate its performance.

---

## Objectives
- Read the dataset from a CSV file
- Understand the dataset structure
- Perform exploratory data analysis
- Visualize class distribution
- Train a Random Forest model
- Evaluate the model performance
- Analyze feature importance

---

## Dataset Information
The Iris dataset contains **150 samples** and **5 columns**:

### Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target
- Species

### Classes
- Setosa
- Versicolor
- Virginica
